# Character Portrait Generation Prompts

Use these prompts with ChatGPT/DALL-E to generate character portraits.
Save each image as a .webp file with the character's name (lowercase).

## Instructions
1. Go to ChatGPT and use DALL-E
2. Copy each prompt below
3. Download the generated image
4. Convert to WebP format (or save as .webp)
5. Rename to match the filename below
6. Place in this folder (public/portraits/)

## Style Notes
- All portraits should be 1:1 square aspect ratio
- Target size: 512x512 or higher
- Consistent sepia/noir color grading
- Dramatic chiaroscuro lighting
- Painterly, oil painting aesthetic
- 1920s period-accurate clothing and styling
