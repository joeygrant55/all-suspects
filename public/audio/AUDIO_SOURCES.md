# Audio Sources for All Suspects

Place audio files in this directory with the exact filenames listed below.

## Required Files

### Background Music
- **jazz-ambient.mp3** - 1920s jazz ambient loop
  - Generate via Suno.ai: "1920s speakeasy jazz, ambient, lo-fi, noir mystery, soft piano and muted trumpet, seamless loop, moody atmosphere"
  - Or use royalty-free from: Pixabay, Freesound, or Uppbeat

### Room Ambience (looping)
- **ambience-fireplace.mp3** - Crackling fire, warm room
  - Freesound.org search: "fireplace ambience"
- **ambience-clock.mp3** - Grandfather clock ticking, quiet room
  - Freesound.org search: "clock tick ambience"
- **ambience-wind.mp3** - Wind outside, drafty manor
  - Freesound.org search: "wind ambience indoor"
- **ambience-kitchen.mp3** - Subtle kitchen sounds, pots settling
  - Freesound.org search: "kitchen ambience quiet"

### Sound Effects (one-shot)
- **sfx-footstep.mp3** - Single wooden floor footstep
  - Freesound.org search: "footstep wood floor"
- **sfx-click.mp3** - UI button click, subtle
  - Freesound.org search: "ui click subtle"
- **sfx-evidence.mp3** - Discovery chime/revelation sound
  - Freesound.org search: "mystery discovery chime"
- **sfx-door.mp3** - Door creaking open
  - Freesound.org search: "door creak open"
- **sfx-conversation.mp3** - Paper rustling or attention sound
  - Freesound.org search: "paper rustle"

## Recommended Sources

1. **Suno.ai** (AI Music Generation)
   - Best for background music
   - Free tier available
   - Use prompts like: "1920s jazz, noir mystery, ambient loop"

2. **Freesound.org** (Free SFX)
   - Huge library of CC0/CC-BY sounds
   - Great for ambience and SFX
   - Always check license

3. **Pixabay** (Royalty-Free)
   - Music and SFX
   - Free for commercial use
   - No attribution required

4. **Uppbeat** (Royalty-Free Music)
   - Curated music library
   - Free tier with attribution

## Audio Specs

- **Format**: MP3 (widely supported)
- **Sample Rate**: 44.1kHz
- **Bit Rate**: 128-192kbps (good quality, reasonable size)
- **Ambience/Music**: Should loop seamlessly
- **SFX**: Short, punchy, no silence padding

## Quick Setup

If you want to quickly test the audio system without gathering all files,
you can start with just:
1. jazz-ambient.mp3 - Background music
2. ambience-fireplace.mp3 - Test room ambience
3. sfx-click.mp3 - Test UI feedback

The system gracefully handles missing files (silent fallback).
